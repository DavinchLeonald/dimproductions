Game Manual

Connection & Start
Press Start to request MIDI access. Once connected, use your MIDI keyboard to rapidly tap the required notes (see Modes below). Before starting a run, test which physical key on your keyboard sends C (and D if needed).

Modes

  • Erlkönig (C): Tap C only.
  • Diavolo (C–D): Alternate strictly between C and D.

How to Play
After you press START, a short countdown begins, then a 10-second burst is measured. Hit as fast and as accurately as you can within the time limit.

Results
You’ll see: HITS (total), SPEED (hits/s), STABILITY (timing consistency), and SCORE (speed × stability).

Shortcuts
Space = Start/Stop,  M = Mute,  ←/→ = Mode toggle.

Latency Tips
Avoid Bluetooth audio (high latency). Use a wired output when possible.

Browser
Chrome (or Chromium-based) is recommended.
